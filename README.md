## 📗 説明
シンプルで使いやすいToDoアプリケーションです。 

Cloud Firestoreのリアルタイムアップデート利用しているため、データ更新の高速であるのが特徴です。

## 💻 使用技術
- Nuxt.js
- Vue.js
- TailwindCSS
- Firebase Authentication
- Firebase Hosting- 
- Cloud Firestore
